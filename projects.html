Annual clock

I developed an visual annual clock application as my thesis during my internship at Kaakkois-Suomen Tieto Oy.

It was developed in Visual Studio -programming environment using PHP programing language, HTML, JavaScript, CSS and Scalable Vector Graphics. The application was focused on creating an application which can be used on both computers and tablets and is able to interact with Microsoft Office 365 Outlook calendars. I wrote my thesis in finnish and here is a link to my thesis about this application: Visual clock.
